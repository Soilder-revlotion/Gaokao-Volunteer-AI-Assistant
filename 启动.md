# 启动指南

## 前置条件

- Python 3.12+ 已安装
- Ollama 已安装，qwen2.5:3b 模型已下载
- Python 依赖已安装（chromadb / sentence-transformers / flask / requests）

## 启动步骤

### 1. 启动 Ollama

打开一个 CMD 窗口：

```bash
ollama serve
```

保持窗口不要关闭。

### 2. 启动项目

打开另一个 CMD 窗口：

```bash
cd D:\Holiday\claude1\高考志愿AI助手
py scripts\05_web.py
```

等待出现 `访问地址: http://localhost:5000` 即启动成功。

### 3. 打开浏览器

访问 [http://localhost:5000](http://localhost:5000)

## 停止

在对应窗口按 `Ctrl + C` 即可停止。
